# Пост в канал (формат «РАБОТЫ»)

---

**РАБОТЫ**

🔹 **_InsrumentsBOT — Telegram‑хаб для SMM‑специалистов_**

> Коммерческий заказ: инструмент для **ведения каналов и оформления постов** — посты, кнопки, премиум‑эмодзи, скачивание видео, кружки, отложка. **Передан заказчику.**

🔌 **_Что внутри:_**

🟣 хаб: видео/MKV → кружок, каналы, publisher‑боты по токенам, MTProto  
🟣 постер: переслал контент → черновик → канал (текст, фото, видео, альбомы)  
🟣 скачать видео: YouTube, VK, RuTube, TikTok  
🟣 URL‑кнопки с цветами, премиум‑эмодзи, «картинка из эмодзи»  
🟣 отложенная публикация, таймзона канала, настройки форматирования  
🟣 деплой на VPS (systemd)  

🔌 **_Стек: 🐍_**

➤ <u>Python, aiogram 2, Telethon, ffmpeg, JSON state, Ubuntu/systemd</u> ◀

⚠️ Коммерческий проект — **исходники не публикуются**. В репозитории описание и скрины.
